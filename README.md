# KINDBRIDGE-Non-Profit-Engagement-App-
KINDBRIDGE is a UI/UX design project focused on creating a high-fidelity mobile application that connects users with social causes through donations, volunteering, and community events. It provides a unified platform to discover, contribute, and engage with meaningful initiatives.
